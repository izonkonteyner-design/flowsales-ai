# Product Roadmap

This roadmap reflects the current rebuild direction of FlowSales AI.

## Phase 1: Foundation

- Establish the app shell
- Define shared UI primitives
- Add Supabase client and server helpers
- Create validation schemas and demo data
- Restore protected routes and auth surfaces

## Phase 2: CRM workflows

- Leads list and lead detail
- Products catalog
- Quotes list, detail, and creation
- Tasks board
- Activity timeline surfaces

## Phase 3: Intelligence

- AI assistant workspace
- Knowledge document ingestion
- Suggested follow-ups and automation hooks
- Context-aware quoting and lead summaries

## Phase 4: Operations

- Reports and KPI surfacing
- Subscription and plan management
- Settings and workspace administration
- CI, documentation, and deployment hardening

## Phase 5: Production readiness

- Live Supabase integrations
- Stronger automated tests
- End-to-end browser coverage
- Monitoring and error tracking
- Payment and AI provider wiring

## Guiding principles

- Keep the UI premium but maintainable
- Prefer tenant-safe defaults
- Preserve demo mode for local development
- Add live integrations behind clear feature boundaries
